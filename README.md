# use-whispers

An agent skill for building products and integrations on top of
[Southwind Whispers](https://app.southwind.ai), an AI-powered analytics and
report generation platform.

## What's Included

```text
skills/use-whispers/
├── SKILL.md       # Integration guide, workflows, and guardrails
├── reference.md   # Curated API endpoint catalog
├── VERSION
└── CHANGELOG.md
```

## What It Helps With

- Connect files, documents, and external REST APIs as data sources.
- Generate reports with `custom_report`, `rover_report`, or `blank_report`.
- Poll or stream async report progress.
- Render BlockNote report blocks, including ECharts chart blocks.
- Use report chat, public sharing, and export workflows.

## Getting Started

1. Install `skills/use-whispers` in your agent environment.
2. Create an API key at [app.southwind.ai/settings/api-keys](https://app.southwind.ai/settings/api-keys).
3. Ask your agent to build, debug, or extend a Whispers integration.
